# Getting Started

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/sridhar7601/react-frontend.git
2. Navigate into the project directory:

   ```bash
   cd react-frontend
3. Install dependencies:(Use either npm or yarn):

   ```bash
   npm install
   #or
   yarn install

4. Run the code:
    ```bash
    npm run dev
